# HTML Email Template — DataPulse Newsletter

A responsive promotional HTML email template built using table-based HTML and inline CSS, following industry-standard email development practices.

## Live Preview
[View Live](https://yourusername.github.io/html-email-template)

## About the Project
This project is a fully structured HTML email template for a fictional analytics newsletter brand called **DataPulse**. It was built to demonstrate core email development skills including table-based layout, inline CSS styling, and responsive design for email clients.

## Features
- Header with brand logo and tagline
- Hero section with headline, subtext, and CTA button
- 2-column feature block with article cards
- Stats banner with key metrics
- Footer with social links, unsubscribe, and privacy policy

## Tech Stack
- HTML5
- CSS (inline styles — no external stylesheets)
- Table-based layout (email client compatible)

## Email Client Compatibility
Tested and compatible with:
- Gmail (web & mobile)
- Outlook
- Apple Mail
